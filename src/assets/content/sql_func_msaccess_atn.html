Example

Return the arc tangent of a number:

SELECT Atn(5) AS AtnNum;
Try it Yourself »

Definition and Usage

The Atn() function returns the arc tangent of a number.

Syntax

Atn(number)

Parameter Values

Parameter Description
number Required. A numeric value

Technical Details

Works in: From Access 2000
