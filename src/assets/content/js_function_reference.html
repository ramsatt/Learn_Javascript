Function Object Methods & Properties

Name Description
apply() Calls a function with a specified this value and an array of arguments
bind() Returns a function with a specified this value
call() Calls a function with a specified this value and individual arguments
length Returns the number of (expected) parameters of a function
name Returns the name of the function
toString() Returns the name of the function

this Precedence

Use the following precedence of order to determine which object this refers to:

OrderObjectBecause
1bind() this is in a function being called using bind()
2apply() this is in a function being called using apply()
2call() this in is a function being called using call()
3Object method this in is an object function (method)
4Global scope this in isa function in the global scope
