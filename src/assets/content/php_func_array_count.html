Definition and Usage

The count() function returns the number of elements in an array or in a countable object.

Syntax

count(value, mode)

Parameter Values

Parameter Description
value Required. Specifies an array or countable object
mode Optional. Specifies the mode. Possible values:
  • 0 - Default. Does not count all elements of multidimensional arrays
  • 1 - Counts the array recursively (counts all the elements of multidimensional arrays)

Technical Details

Return Value: Returns the number of elements in the array
PHP Version: 4+
Changelog: PHP 8.0 - Will throw TypeError on invalid countable types passed to the value parameter.

More Examples

Example

Count the array recursively:

<?php
$cars=array
  (
  "Volvo"=>array
  (
  "XC60",
  "XC90"
  ),
  "BMW"=>array
  (
  "X3",
  "X5"
  ),
  "Toyota"=>array
  (
  "Highlander"
  )
  );

echo "Normal count: " . count($cars)."<br>";
echo "Recursive count: " . count($cars,1);
?>
