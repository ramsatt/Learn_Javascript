Definition and Usage

The ATAN2() function returns the arc tangent of two numbers.

Syntax

ATAN2(a, b)

Parameter Values

Parameter Description
a, b Required. Two numeric values to calculate the arc tangent of

Technical Details

Works in: From MySQL 4.0
