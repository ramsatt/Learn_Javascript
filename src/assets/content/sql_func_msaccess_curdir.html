MS Access CurDir() Function

❮ Previous ❮ MS Access Functions Next ❯

Example

CurDir()
CurDir("D")

Result:

C:\Documents and Settings\user\My Documents
D:\

Definition and Usage

The CurDir() function returns the full path for a specified drive.

Syntax

CurDir(drive)

Parameter Values

Parameter Description
drive Optional. The drive to return the full path for. If omitted, the current drive is used

Technical Details

Works in: From Access 2000
